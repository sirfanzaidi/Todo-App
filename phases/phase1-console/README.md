# Phase 1: Console Todo App

A simple command-line todo application implemented in Python.

## Features
- Add, view, update, and delete todos
- Console-based interface
- Local data storage

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
python src/todo_app.py
```

## Testing

```bash
# Run tests
pytest tests/
```
